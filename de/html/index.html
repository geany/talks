Was kann Geany
besonderes?

Meta

  • Editor mit Funktionen einer integrierten Entwicklungsumgebung
  • Seit 2005 in Entwicklung
  • Aktuelle Version 1.23.1
  • Entwicklungsziele
    • Geringe Systemanforderungen
    • Wenige zusätzliche Abhängigkeiten
  • Geschrieben in C/C++
  • Basiert auf Scintilla
  • Lizenz GPLv2+

Visualisierung

  • Syntaxhervorhebung für verschiedenste Dateitypen (ca. 50)
  • Markierung von Fehlern beim Übersetzen

Entwicklungsunterstützung

  • Anpassbares Erstellen-Menü
  • Erweiterungen für Debugging, Versionskontrolle, Webseitenvorschau, Hilfe ...
  • Autovervollständigung von bekannten Symbolen/Variablen/Makros/...
  • Vorlagen und Code-Schnipsel

Quellcode-Navigation

  • Symbolbrowser zum einfachen Navigieren innerhalb eines Dokumentes
  • Funktion zum Springen zur Definition/Deklaration eines Codeblocks
  • Öffner einer "verlinkten" Datei über das Kontextmenü

Sonst so

  • Plattformunabhängig: (Linux, Unix, Windows)
  • Erweiterbar: Pluginschnittstelle
  • und noch ein paar mehr ...

Wer entwickelt
Geany?

  • Kern-Team: ca. 5 Mann
  • Contributoren (Quellcode):
    • Geany: ca. 50 im letzten Jahr
    • Geany-Plugins: ca. 20 im letzten Jahr
  • Übersetzer: ca. 30

Gibt es andere
Themes?

Theming

  • Geany nutzt die .gtkrc-Einstellungen
  • Verschiedene Themes für das Editorfenster gibt es beim Geany-Themes-Projekt
  • Themes sind kleine Textdateien: einfach zu erstellen und installieren (a.k.a. kopieren)
  • Darüber hinaus sind eigene Anpassungen recht einfach möglich
  • Einfaches Wechseln zwischen Themes

Wann gibt es das nächste Release

  • Version 1.24: Zweites April-Wochenende (vorraussichtlich)
  • Gemeinsames Release von Geany und den Geany-Plugins

Neuigkeiten für Geany

  • Unterstützung für Gtk3
  • Unterstützung für Ctags und Vi-Tag-Dateien
  • Möglichkeit, Plugin-Tastenkürzel direkt von der Pluginverwaltung aus zu konfigurieren
  • Update von Scintilla
  • Viele neue Dateitypen (z.B. CUDE, Rust, Batch) und Verbesserungen an bereits bekannten

... und für Geany-Plugins

  • Unterstützung von Gtk3
  • Neue Plugins. z.B.
    • GeanyPy -- Pluginschnittstelle in Python
    • PairTagHighlighter -- Finden von zusammengehörigen Tags
    • PoHelper -- Unterstützung für gettext-Dateien
  • Viele kleine Updates wie z.B. Support für Subversion 1.7 in GeanyVC

Autovervollständigung

  • Basierend auf:
    • Statischen Tag-Files
    • Erkennen von Variablen und oft verwendeten "Wörtern" in der Sitzung
    • Keine kontextsensitive Erkennung wie bei Eclipse

Wie kann ich ...

übersetzen

Erstellen-Menü

  • Einfache Nutzung über Menü oder Tastatur
  • Grundbefehle wie Übersetzen oder Ausführen für viele Sprachen vorhanden
  • Verzicht auf komplexe Konstrukte wie "alles übersetzen" → Makefile
  • Konfigurierbar global und pro Dateityp
